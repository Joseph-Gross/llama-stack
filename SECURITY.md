# Security Policy

## Supported Versions

We currently support the following versions with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Security Features

Llama Stack includes several security features:

- **Content Safety**: Shield system for filtering unsafe content
- **Code Scanning**: Detection of insecure code patterns
- **Prompt Injection Protection**: Detection of malicious prompts
- **Rate Limiting**: Protection against DoS attacks
- **Input Validation**: Schema validation for requests
- **API Key Security**: Secure API key handling
- **Security Headers**: Protection against common web vulnerabilities

## Security Best Practices

See our [Security Best Practices](docs/source/security/security_best_practices.md) guide for recommendations on securing your Llama Stack deployment.

## Reporting a Vulnerability

Please report vulnerabilities to our bug bounty program at https://bugbounty.meta.com/

For non-critical security issues, please open an issue on GitHub.
